# Лабораторна робота №4 – Mobile Application

## Тема
**Розробка мобільного застосунку “Форма опитування”**  

Завдання лабораторної роботи передбачає створення мобільного застосунку для ОС сімейства **Android** (можна запускати на фізичному чи віртуальному пристрої).  
Результатом виконання роботи є програмне забезпечення (застосунок), який реалізує функції згідно з варіантом завдання та має якісний інтерфейс користувача.

---

## Мета роботи
- Розробити застосунок для проведення опитування користувача.
- Відображати користувачу питання по черзі.
- Приймати відповіді через текстове поле.
- Зберігати відповіді у файл на диску.
- Забезпечити зрозумілий і простий інтерфейс користувача.

---

## Використані технології
- **Python 3.9+**
- **Flet** (Flutter UI для Python)
- Тестування: запуск на десктопі та на мобільному емуляторі Android (Flet дозволяє це через браузер або вебview)

---

## Структура проєкту
