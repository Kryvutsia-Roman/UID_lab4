# lab4

## Опис проєкту
Цей проєкт був розроблений у рамках лабораторної роботи №4 з курсу **«Розробка інтерфейсів користувача»**.

Застосунок реалізує просту **форму опитування**, яка послідовно відображає користувачу запитання, дозволяє вводити відповіді та зберігає їх у текстовий файл на пристрої.

Проєкт створений з використанням фреймворку Flutter та мови програмування Dart.

---

## Мета роботи
Ознайомитися з принципами створення мобільних застосунків за допомогою Flutter, навчитися створювати графічний інтерфейс користувача, обробляти події користувача та зберігати дані у файловій системі.

---

## Використані технології
- **Flutter**
- **Dart**
- **Material Design**
- **File System API**

---

## Функціонал застосунку
- Відображення послідовності запитань
- Введення відповіді у текстове поле
- Перевірка введення (порожня відповідь)
- Перехід до наступного питання
- Повідомлення про завершення опитування
- Збереження відповідей у текстовий файл

- **MaterialApp** — головний віджет застосунку  
- **Scaffold** — структура екрана

- **Text** — відображення питань

## Структура файлів
Проєкт має модульну структуру для кращої організації коду:

- **lib/main.dart**: Точка входу в додаток. Ініціалізує `MaterialApp` з темою та встановлює `SurveyPage` як головний екран.
- **lib/screens/**: Папка з екранами додатку.
  - **survey_page.dart**: Головний екран опитування. Відображає поточне запитання, обробляє введення відповіді користувачем, перевіряє валідність, переходить до наступного запитання та показує діалог завершення. Використовує `SurveyModel` для запитань та `FileService` для збереження.
- **lib/models/**: Папка з моделями даних.
  - **survey_model.dart**: Містить статичний список запитань для опитування.
- **lib/services/**: Папка з сервісами для бізнес-логіки.
  - **file_service.dart**: Відповідає за збереження відповідей у файл `/storage/emulated/0/Download/answers.txt` на пристрої.
